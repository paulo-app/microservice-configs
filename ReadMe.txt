Esta carpera local "microservice-configs" corresponde al respositorio de configuraciones cetralizado.
En GitHub.com, crear un repositorio nuevo con el mismo nombre, o sea "microservice-configs"

Luego, desde una consola desde la carpeta local "microservice-configs" ejecutar:
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/paulo-app/microservice-configs.git
git push -u origin main
